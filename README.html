01 — Features

What it does.

Core Interactions & Design Decisions

01

Editorial Grid

An asymmetric 4-column CSS Grid on the homepage. Images intentionally break the baseline to create visual tension — controlled chaos, like a printed magazine spread.

CSS Grid
02

Raw Archive

A growing collection of photographs that load in batches of 8–15 as you scroll — no dump, just discovery. Full drag & drop logic written from scratch — two-click zoom system, z-index management, drop rotation.

Vanilla JS
03

Frosted Glass UI

Text containers use backdrop-filter: blur() to float above imagery. Layers of transparency create a tactile, physical depth on a flat screen.

CSS Effects
04

Custom Cursor

A dual-element cursor — a dot and a ring — that expands and fills solid petrol on hover over links, cards, and h1 headings. Smooth via the Web Animations API.

Web Animations API
05

Scroll Animations

Staggered fadeUp keyframes across every page. Each element enters with a cubic-bezier spring — fast out, slow settle. No libraries needed.

CSS Keyframes
06

Typography Clash

Playfair Display (emotion, editorial) paired with Fira Code (logic, system). One serif, one monospace. Two voices in constant dialogue across every page.

Google Fonts

04 — Setup

04 / 05

Quick Start.

Running locally in 60 seconds

01

Clone the Repo

Download or clone the repository. All assets live in the Fotos/ folder alongside the HTML files.

02

No Build Step

No npm, no webpack, no bundler. This project runs entirely in the browser — zero dependencies to install.

03

Open with Live Server

Use the VS Code Live Server extension or any local server. Opening index.html directly may block font requests in some browsers.

04

Deploy via GitHub Pages

Push to GitHub, enable Pages under Settings → Pages, set source to main branch. Live in under a minute.

// Image Credits

Photos sourced via Unsplash. All photographers are credited inline per archive item (e.g., #001 // HANEN SOUHAIL). Please review Unsplash's license terms before redistributing.

// File Paths on Windows vs. Unix

A few image paths in archive.html use backslashes (Fotos\). These work on Windows but will break on macOS/Linux and GitHub Pages. Replace with forward slashes (Fotos/) before deploying.

The Studio.

Isabelle Striewski — Design & Development

"The goal was to bridge the gap between rigid data and fluid aesthetics."

STUDIO 0x is a personal project exploring the intersection of editorial design and modern web interfaces. Every decision — from the asymmetric grid to the drag-and-drop archive — was intentional. No template, no UI kit, no AI-generated layout. Just a blank stylesheet and a vision.

The name carries a dual meaning: Studio as a creative space, 0x as a hexadecimal prefix — the language of machines. Fashion meets code.

Design Principles.

01 Asymmetry over convention — break the grid to guide the eye.
02 Active whitespace — empty areas are deliberate, not lazy.
03 Layering for depth — glass over image, text over space.
04 Typographic tension — serif emotion vs. monospace logic.
